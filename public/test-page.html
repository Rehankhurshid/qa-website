Setup Instructions:

  1. Start the development server: cd qa-detector && npm run dev
  2. Visit http://localhost:3000 and sign in with an @activeset.co email
  3. Create a new project in the dashboard
  4. Copy the embed token from the project
  5. Replace YOUR_TOKEN_HERE in the script tag below with your actual token
  6. Open this HTML file in a browser (double-click or use open test-page.html)
